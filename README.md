# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a simple friendship tracker app made with React.  
It helps users keep track of their friends, check when they last contacted them, and log quick interactions like call, text, or video.

## 📌 Project Description

This project was built for Assignment 7.  
The main goal of this app is to help users manage friendships in a simple and organized way.

Users can:
- see all friends in cards
- open a friend details page
- log call, text, or video interactions
- view all interaction history in timeline
- see friendship analytics in a pie chart

## ✨ Main Features

### 1. Friend Management
The home page shows all friends in a responsive card layout.  
Each card includes profile picture, name, tags, status, and days since last contact.

### 2. Quick Check-In System
Inside the friend details page, users can click **Call**, **Text**, or **Video** buttons.  
When clicked, a new timeline entry is added automatically and a toast notification appears [web:32][web:39].

### 3. Timeline and Stats
The timeline page shows the history of all interactions.  
The stats page uses a pie chart to show total Call, Text, and Video interactions visually [web:38].
